# 장기적 가챠 정책!

## 게임 정보
- **장르**: 가챠, 파라미터 관리
- **플랫폼**: 모바일
- **게임 엔진**: Unity
- **개발 기간**: 2018. 12. ~ 2019. 2. (3개월)

## 개요
"장기적 가챠 정책!"은 광기 어린 가챠 게임으로, 유저들에게 어마무시한 돈을 쓰게 만드는 현대의 가챠 게임 트렌드를 풍자하는 목적으로 만들어졌다.
세계정부의 대통령이 되어 국민들의 장기를 팔아 가챠 자금을 마련하고, 0.0001%의 확률을 뚫고 영광스러운 새 "SSSSR" 등급 카드를 획득하자!
